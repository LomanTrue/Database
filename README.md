# Database

Realization of dalabase with SQL.

# Configuration

'config.txt' has SQL query that you can modify.

There is a list of supported commans:
- SELECT
- FROM
- WHERE
- (LEFT|RIGHT|INNER)JOIN
- CREATE TABLE
- DROP TABLE
- AND
- OR
- IS
- NOT
- NULL
- ON
- UPDATE
- INSERT
- VALUES
- DELETE
- PRIMARY KEY
